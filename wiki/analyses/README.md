# Analyses

This directory holds durable outputs produced from questions, comparisons, timelines, and syntheses.

Examples:

- a comparison between two frameworks
- a timeline assembled from many sources
- an argument map
- a reading memo
- a trip plan
- a due diligence summary

If an answer would be useful later, file it here instead of leaving it only in chat history.
