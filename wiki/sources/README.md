# Sources

Create one page per ingested source unless multiple files are intentionally treated as one bundle.

Suggested sections:

- Source details
- Summary
- Key claims
- Notable evidence
- Connections to existing pages
- Contradictions or tensions
- Follow-up questions

Each source page should point to the raw file path and link to impacted entity and concept pages.
